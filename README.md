content-s3-upload
=================

front end for uploading files directly to s3

create a .env file with your s3 id and secret
{
    "S3_ID": "***",
    "S3_SECRET": "***/***"
}

install nodemon

`npm install -g nodemon`

install dependencies

`npm install && bower install`

launch server

`nodemon app.js`

open browser

http://localhost:3000
